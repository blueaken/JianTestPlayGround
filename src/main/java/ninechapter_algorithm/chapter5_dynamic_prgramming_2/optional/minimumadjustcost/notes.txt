res[i][j]: 当有前i 个数组元素时，转化成value j 所需要的最小调整代价


方程： res[i][j] = Math.min((Math.abs(j - A.get(i)) + res[i-1][k]), res[i][j])， ｜j － k｜ <= target,
      k是前面一个值，取最小值是因为对 ｜j － k｜ <= target， 会有多个k满足条件

init: 第一行为第一个树调整到j所需的代价

结果：最后一列中最小值 MIN(res[n-1][x])
