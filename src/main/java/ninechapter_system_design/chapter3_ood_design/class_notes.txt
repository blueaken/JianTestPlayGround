OOD 大法：
1. 想想有什么实体？（来源于生活但是高于）
2. 想每个实体有什么属性，包含什么？
3. 想每个实体可以干什么？
4. 怎么贯穿整个流程？ Scenario -第一节课的SNAK大法

note:
- OO Design不要多想算法，简单work就行
- OO Design Manager拥有上帝之眼，调控全局。这样把功能和逻辑分开，把功能集中在实体类中，业务逻辑集中在Manager类中，使业务逻辑的改变不
会影响实体类。
- OO Design 根据不同场景说理，有理就好，没有正确答案. 题可以不一定全对，工程的思想，思考思路最重要
- OO Design 不是逻辑算法， 是信息传递流设计
